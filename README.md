# SAP IS-U Meter-to-Cash QA Portfolio

## Overview
This portfolio simulates how QA supports a utilities transformation programme with SAP IS-U style workflows.

It focuses on the meter-to-cash lifecycle:

**Meter Reading → Device Management → Billing → Invoice → Customer Account / Payment → Reconciliation**

## Why this portfolio matters
This project is designed to show recruiter-facing evidence that I understand the core testing approach around utilities systems, integration-heavy workflows, and end-to-end data validation.

## Scope covered
- Meter reading validation
- Device lifecycle and exchange scenarios
- Billing and invoice validation
- Customer account lifecycle checks
- Error handling, migration, cutover, and go-live smoke coverage
- End-to-end meter-to-cash reconciliation
